# Лабораторная работа №4. Распараллеливание операций над векторами

## Описание работы
Реализовать и распараллелить с помощью технологии OpenMP различные операции над векторами размерности N.

Вариант: 9

N = 11

Тип элемента вектора: Беззнаковый целый

Тип операции: Сложение векторов (a[i]=b[i]+c[i])

## Описание технологий для реализации программы
**#pragma omp parallel for** - это директива OpenMP, которая автоматически распараллеливает цикл for, распределяя итерации между несколькими потоками.

Эта директива опасна в использовании в случае гонки данных, но в операции сложения векторов этого нет, так как итерации независимы и каждый поток пишет изменения в свои элементы. Для других операций может понадобиться использование параметров этой директивы для безопасности выполнения.
