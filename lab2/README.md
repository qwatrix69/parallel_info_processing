# Лабораторная работа №2. Распределенные вычисления

## Описание работы
Сгенерировать в каждом из N процессов вектор чисел. Размерность вектора – М. Произвести поэлементную обработку всех векторов и поместить результирующий вектор в каком-либо процессе. В работе использовать средства MPI для организации распределенных вычислений.

Вариант: 9

N = 6

M = 40

Тип элемента вектора: Беззнаковый целый

Тип поэлементной обработки: Двоичное И

## Описание технологий для реализации программы
**1. MPI_Init(&argc, &argv)** - инициализация окружения MPI

**2. MPI_Comm_rank(MPI_COMM_WORLD, &rank)** - возвращает номер процесса в рамках коммуникатора

**3. MPI_Comm_size(MPI_COMM_WORLD, &size)** - возвращает общее количество процессов в рамках коммуникатора

**4. MPI_Reduce(local_vector, result_vector, M, MPI_UNSIGNED, MPI_BAND, 0, MPI_COMM_WORLD)** - выполненение глобальной операции (редукции) с операцией двоичного И. Массив размером M из local_vector (отправляемый буфер) отправляется в result_vector (принимающий вектор). К каждому элементу массива типа MPI_UNSIGNED применяется операция MPI_BAND.

**5. MPI_Finalize()** - завершение работы окружения MPI
